# Design Rules — 추호승 포트폴리오

레퍼런스: SEGMINT 2023 포스터 / Blade Runner ID 카드 / Bold Typography 그래픽

---

## 핵심 방향

현재 글래스모피즘 + 보라/파랑 그라디언트 디자인은 레퍼런스와 다르다.
목표: **레트로-디지털 / 사이버펑크 / 픽셀 미학** — 날것의 디지털 감각, 강한 타이포그래피, 고채도 컬러 블록.

---

## 컬러 팔레트

| 역할 | 색상 | 용도 |
|------|------|------|
| Primary | `#0033FF` (일렉트릭 블루) | 배경 블록, 강조 영역 |
| Background | `#0a0a0a` (거의 검정) | 기본 페이지 배경 |
| Surface | `#111827` | 카드/패널 배경 |
| Accent | `#00FFAA` (사이버 그린) or `#00CFFF` (사이버 시안) | 포인트, 테두리, 커서 |
| Text Primary | `#FFFFFF` | 제목, 본문 |
| Text Secondary | `#7fa8b8` | 부가 정보, 레이블 |
| Danger/Alert | `#FF2200` | 경고성 요소, 포인트 강조 |

그라디언트는 최소화. 색상은 flat하게, 블록으로 사용.

---

## 타이포그래피

### 폰트
- **디스플레이/헤딩**: 픽셀 폰트 또는 모노스페이스 (예: `JetBrains Mono`, `Space Mono`, `VT323`, `Press Start 2P`)
- **본문**: 모노스페이스 또는 시스템 sans-serif — `JetBrains Mono` 권장
- 일반 sans-serif(Geist 등)는 최소화

### 사이즈 & 스타일
- Hero 제목: `clamp(4rem, 12vw, 10rem)` — 화면을 압도하는 크기
- 섹션 레이블: 소문자 + `letter-spacing: 0.2em` + 픽셀 스타일
- 본문: 작고 촘촘하게 (`text-sm`, `leading-tight`)
- 모든 캡션/메타 텍스트: 대문자 (`uppercase`)

### 원칙
- 텍스트 자체가 비주얼 요소 — 큰 글자를 화면에 꽉 채운다
- 줄바꿈을 의도적으로 배치해 레이아웃 구성
- 영문은 ALL CAPS 또는 소문자, 혼용 최소화

---

## 레이아웃

- **그리드 기반**: 12컬럼 또는 명확한 블록 분할
- **풀블리드 섹션**: 섹션마다 배경색 블록으로 강하게 구분 (SEGMINT의 블루/그레이 반반 분할 참고)
- **여백**: 최소화하거나 극대화 — 중간 없음. 꽉 차거나 비어있거나.
- **UI 프레임 요소**: 터미널 창, 브라우저 크롬, ID 카드 같은 UI 메타포 활용 (Blade Runner / Help! 참고)
- **픽셀/글리치 디테일**: 경계선, 구분선, 배경 텍스처에 픽셀 패턴 사용

---

## 컴포넌트 스타일

### 카드
- 배경: `#111827` 또는 `#0033FF`
- 테두리: `1px solid #00CFFF` 또는 `1px solid #0033FF`
- 모서리: `rounded-none` (직각) 또는 `rounded-sm` (최소)
- hover: 테두리 색 변경 + 미세 glitch 효과

### 버튼
- 배경 채움형: 블루 블록 (`bg-[#0033FF]`), 대문자 텍스트
- 아웃라인형: 얇은 테두리 + 모노스페이스 텍스트
- hover: 색상 반전 (배경↔텍스트)
- 그라디언트 버튼 없음

### 섹션 레이블
```
[ ABOUT ] / [ SKILLS ] / [ WORK ]
```
대괄호로 감싼 터미널 스타일 레이블 사용

### 구분선
- `border-t border-[#0033FF]` 또는 픽셀 도트 패턴

---

## 애니메이션

- 부드러운 fade/blur 대신 **glitch**, **scanline**, **타이핑 효과** 우선
- CSS `@keyframes`로 구현, 라이브러리 의존 최소화
- 스크롤 애니메이션: subtle translate (translateY 10px 수준), opacity 변화
- 마우스 hover: 즉각적 전환 (`transition-none` 또는 `duration-75`)
- 배경에 CRT scanline 오버레이 텍스처 (pseudo-element로)

---

## 금지 사항

- 소프트 그라디언트 배경 (현재 보라/파랑 blob 제거)
- backdrop-blur 글래스모피즘
- 둥근 카드 (`rounded-xl`, `rounded-2xl`)
- 파스텔/뮤트 컬러
- 장식적 이모지

---

## 현재 디자인과의 차이

| 현재 | 목표 |
|------|------|
| 보라/파랑 그라디언트 | 일렉트릭 블루 플랫 블록 |
| 글래스모피즘 카드 | 직각 터미널 카드 |
| 부드러운 애니메이션 | 글리치/타이핑 효과 |
| Geist Sans | Space Mono / VT323 |
| 둥근 모서리 | 직각 또는 최소 라운드 |
| 그라디언트 텍스트 | 흰색 or 사이버 그린 단색 |
